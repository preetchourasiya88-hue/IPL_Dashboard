🏏 IPL Dashboard Project Explanation – Excel
📌 Project Overview

I created an interactive IPL Dashboard in Excel to analyze Indian Premier League match data, team performance, player statistics, and tournament insights.
The main objective of this project was to convert raw IPL data into meaningful visual insights using Excel tools like Pivot Tables, Charts, Slicers, KPI Cards, and Conditional Formatting.
This dashboard helps users quickly understand:

-> Team performance
-> Player statistics
-> Match results
-> Winning trends
-> Tournament insights
It provides all important IPL analytics in a single interactive dashboard.

📊 KPI (Key Performance Indicators) Used

1. Total Matches
Displays the total number of IPL matches analyzed.

Purpose:
        Helps understand the dataset size and tournament coverage.

2. Total Teams
Shows the total number of teams participating.

Purpose:
        Used to analyze competition diversity.

3. Total Runs Scored
Represents the total runs scored across all matches.

Purpose:
        Measures overall tournament scoring performance.

4. Total Wickets Taken
Displays the total wickets taken in IPL matches.

Purpose:
        Helps analyze bowling effectiveness.

5. Highest Team Score
Shows the highest score achieved by any team.

Purpose:
        Highlights best batting performances.

6. Win Percentage
Displays winning percentage of teams.

Purpose:
        Helps compare team consistency and success rate.

7. Player Performance Metrics
Includes:
-> Most Runs
-> Most Wickets
-> Highest Strike Rate
-> Best Economy Rate

-> Purpose:
        Used to identify top-performing players.

8. Toss Win vs Match Win Analysis
Shows how often toss-winning teams also won matches.

Purpose:
         Helps analyze toss impact on match results.
